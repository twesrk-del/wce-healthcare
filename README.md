# WCE Healthcare Claim Status Checker

แอปพลิเคชันตรวจสอบสถานะการเบิกค่ารักษาพยาบาล WCE HEALTHCARE

## 🏥 Features

### ✅ Employee Panel (ด้านซ้าย)
- ค้นหาด้วย **รหัสพนักงาน** (เช่น W0123) หรือ **ชื่อ-สกุล** (เช่น สมชาย รักดี)
- ดูเฉพาะข้อมูลการเบิกของตัวเอง
- ติดตามสถานะการเบิก พร้อมวันที่และรายละเอียด

### 🔧 Admin Panel (ด้านขวา)
- Login ด้วย **admin** / **1234**
- ดูตารางรวมของพนักงานทั้งหมด
- เพิ่มและลบรายการการเบิก
- จัดการผู้ดูแลระบบเพิ่มเติม

## 📋 ข้อมูลที่เก็บ

- **ชื่อ-สกุล** 
- **รหัสพนักงาน**
- **หน่วยงาน**
- **สาเหตุ/อาการของโรค**
- **ชื่อคลินิก**
- **จำนวนเงินที่สำรองจ่าย**
- **สถานะการเบิก** (Timeline):
  - ✓ รับเอกสาร
  - ✓ ส่งต่อเอกสาร
  - ✓ เบิกจ่ายเรียบร้อย
  - ✗ ไม่คุ้มครอง
  - ? ขอเอกสารเพิ่มเติม

## 🚀 Quick Start

1. เปิด `index.html` ในเบราว์เซอร์
2. **สำหรับพนักงาน**: กรอกรหัสพนักงานหรือชื่อ แล้วกด Check Status
3. **สำหรับ Admin**: กรอก admin / 1234 เพื่อเข้าสู่ระบบจัดการ
